# Smart Grocery & Meal Planner

AI-powered meal planning app with cloud sync.

## Features
- 🥗 Pantry inventory management
- 🤖 AI recipe suggestions (Groq)
- 🔍 Recipe database search (TheMealDB)
- 💾 Cloud sync (Supabase)
- 📋 Smart shopping lists
- 🔄 Recurring items

## Setup

1. Create a free Supabase account at supabase.com
2. Create a new project and run the SQL schema (see supabase-schema.sql)
3. Get your Supabase URL and anon key from Project Settings → API
4. Create a free Groq account at console.groq.com and get an API key
5. Open the app and configure your credentials in Settings

## Technologies
- React
- Tailwind CSS
- Supabase (PostgreSQL)
- Groq AI (LLaMA)
- TheMealDB API
